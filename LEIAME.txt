Modulos:
    ->Foram feitos 3 modulos para esse trabalho, sendo eles:
        - libLista: que manipula minha estrutura de dados. Enfilera (enqueue), desenfilera (dequeue) e imprime os dados.
        - libBeale: que contem funções que checa argumentos de entradas e outras funções como colocar letras em minusculas
            vereficar se um arquivo foi aberto corretamente e outras funções.
        - libEncoder: contem funções que são utilizadas para codificar a mensagem.
        - libDecoder: contem funções que são utilizadas para decodificar a mensagem.

Estrutura de Dados:
    - Para esse trabalho foi usado listas encadeadas para armazenas os dados.
    - Como cada chave tem a possibilidade de ter vários valores, foi utilizado lista de listas.
    - As chaves são umas lista duplamente encadeadas e cada chave tem uma ligação com uma outra lista, a lista  
        de valores correspondente a essa chave.
    - Uma representação da estrutura:
        A = B = C = D = E
        |   |   |   |   |
        0   3   5   9   8
        |   |   |   |   |
        2   6   7   10  16
            |   |       |
            54  12      13

    - A desvantagem dessa estrutura é a busca, pois talvez precise percoreer toda a lista para achar o valor.
    - A vantagem é que é facil de inserir e remover elementos.
